# Dog Breed Classification — EfficientNetV2 + CBAM
This project performs 120-class dog breed classification using the Stanford Dogs Dataset, built with EfficientNetV2 enhanced by CBAM attention for stronger feature extraction and higher accuracy.
## Features
  EfficientNetV2 + CBAM (Channel + Spatial Attention)
  Input size 224×224
  Full training pipeline (data loading → augmentation → training → evaluation)
  Metrics: Accuracy, Top-5 Accuracy, F1-score
  Complete implementation in the notebook
